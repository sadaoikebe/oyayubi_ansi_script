Windows 10 上で ANSI 配列のキーボードで親指シフト入力をするための oyayubiwm スクリプト。

必要なもの
* ANSI 配列のキーボード
* em1keypc
* oyayubiwm

# 配列

  q  w  e  r  t    y  u  i  o  p  [  ]  \

    z  x  c  v  b    n  m  ,  .  /
